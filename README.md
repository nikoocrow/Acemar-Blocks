# Acemar Blocks

Custom Gutenberg blocks for Acemar WordPress theme.

## Blocks

### Hero Slider

Full-width hero slider with:

- Image or video backgrounds (YouTube/Vimeo)
- Customizable content with InnerBlocks
- Left/center/right alignment
- Multiple slides with navigation
- Splide.js integration

## Installation

1. Upload to `/wp-content/plugins/`
2. Install dependencies: `npm install`
3. Build: `npm run build`
4. Activate in WordPress admin

## Development

- `npm start` - Development mode with watch
- `npm run build` - Production build

## Features

- Built with @wordpress/scripts
- SASS for styling
- Splide.js for slider functionality
- Support for images and embedded videos
- Responsive design
- InnerBlocks for flexible content

## Requirements

- WordPress 5.0+
- Node.js 14+
